# 🛒 Uzum Market Copy

Uzum Market saytining asosiy funksiyalarini takrorlovchi onlayn-do'kon interfeysi. Mahsulotlarni ko'rish, kategoriya bo'yicha filtrlash va qidirish imkonini beradi. Loyiha toza **HTML, CSS va JavaScript** asosida yaratilgan, hech qanday framework ishlatilmagan.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

## 🚀 Xususiyatlari

- 🛍️ **Mahsulotlar ro'yxati** — mahsulotlar kartochka (card) ko'rinishida chiroyli chiqariladi
- 📂 **Kategoriyalar bo'yicha filtrlash** — mahsulotlarni kategoriyaga qarab saralash
- 🔍 **Qidiruv (Search)** — mahsulot nomi bo'yicha real vaqtda qidirish
- 💰 **Narx va chegirma** — har bir mahsulotda narx va chegirma foizi ko'rsatiladi
